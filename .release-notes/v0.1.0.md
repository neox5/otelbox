# Release v0.1.0

Initial release of otelbox - a telemetry signal generator for testing observability components.

## Overview

otelbox generates coherent metric signals for testing monitoring pipelines, collectors, and transformation logic without requiring live systems. Built on [simv](https://github.com/neox5/simv) for guaranteed mathematical consistency between related metrics.

## Features

**Core Capabilities:**

- Clock-driven metric generation with guaranteed coherence
- Multiple metric types: counters and gauges
- Transform chains including accumulation and reset behaviors
- Template-based configuration with iterator expansion
- Multi-clock support for different update frequencies

**Configuration System:**

- YAML-based configuration with strict validation
- Template system for generating metric families at scale
- Instance-based entity sharing for coherent metrics
- Iterator patterns for creating hundreds/thousands of series
- Examples included for common patterns

**Export Protocols:**

- Prometheus pull endpoint (HTTP `/metrics`)
- OTLP push via gRPC (port 4317)
- OTLP push via HTTP (port 4318)
- Configurable export intervals and resource attributes

**Operations:**

- Multi-platform binaries (linux, darwin, windows on amd64/arm64)
- Container images for linux/amd64 and linux/arm64
- CLI with serve and validate commands
- Debug logging for troubleshooting
- Reproducible simulations via seed configuration

**Self-Monitoring:**

- Resource monitoring (CPU, memory, goroutines)
- Saturation detection and warnings
- Optional internal metrics exposure

## Installation

### Container (Recommended)

```bash
podman pull ghcr.io/neox5/otelbox:v0.1.0
```

### Binary Releases

Download the latest release for your platform from [GitHub Releases](https://github.com/neox5/otelbox/releases/tag/v0.1.0):

```bash
# Linux (amd64)
wget https://github.com/neox5/otelbox/releases/download/v0.1.0/otelbox-linux-amd64
chmod +x otelbox-linux-amd64
./otelbox-linux-amd64 --version
```

Verify checksums:

```bash
wget https://github.com/neox5/otelbox/releases/download/v0.1.0/otelbox-linux-amd64.sha256
sha256sum -c otelbox-linux-amd64.sha256
```

### From Source

Requires Go 1.25+:

```bash
go install github.com/neox5/otelbox/cmd/otelbox@v0.1.0
```

## Quick Start

```bash
# Run with default configuration
otelbox

# Run with custom configuration
otelbox -config myconfig.yaml

# Enable debug logging
otelbox -config myconfig.yaml --debug
```

Verify metrics:

```bash
curl http://localhost:9090/metrics
```
